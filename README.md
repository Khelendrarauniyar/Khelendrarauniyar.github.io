### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Khelendrarauniyar/Khelendrarauniyar.github.io.git
   cd Khelendrarauniyar.github.io
   ```

2. **Open in browser**

   - Edit `index.html` for content updates

